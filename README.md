# Vertex Alpha Tools
A simple Blender addon that lets you toggle the visualization of a mesh's vertex color alpha as a material overlay, as well the specific alpha values as 3D labels. It also allows you to set individual vertex colors' alpha to specific values.

## Instructions
- Download the repository as a .zip (click in "Code" then "Download Zip")
- In Blender, go to Preferences, Add-ons, Install from Disk.
- Choose the addon zip.
- It's going to be available in the Tools tab, under "Vertex Alpha Tools"
