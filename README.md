# Vertex Color Tools
A simple Blender addon that lets you visualize a mesh's vertex color alpha and also lets you set individual vertex colors' alpha to specific values.

## Instructions
- Download the repository as a .zip (click in "Code" then "Download Zip")
- In Blender, go to Preferences, Add-ons, Install from Disk.
- Choose the addon zip.
- It's going to be available in the Tools tab, under "Vertex Color Tools"
